流水不腐，户枢不蠹，动也

发表于 2020-04-04 更新于 2020-11-01 分类于 生活篇

生命在于运动，快乐来自天然~

20-04-04

因为疫情的原因，已经家里蹲了一个星期了，唯一一次的出门就是遛狗。

之前一直在家里用健身环做运动，跑步是不敢一个人出去跑步的，老师说过两次一起跑，但今天老师问过我的跑步历史，又说了一下他的速度，我惊了，一个小时11公里，我这个菜鸡暂时是别想跟上这速度了，这也成了我下决心今天就出门跑步的原因：尽早开始，加快赶上。

没跑出几百米，我就感觉腿开始变重了QAQ沿着平常去超市的熟悉路线跑了一遭，返程时随意挑了个路口拐进去，想着呈正方形路线运动，应该能回到家。然鹅，转过几次弯之后我感觉自己可能回不到原路了，天色慢慢黑了，跑得也越来越慢，熊略略有些慌。

后来我想，如果下一个路口还是不熟悉的话，就原路返回，或者google地图带我回家，别方。

于是放松下来，发现了路边不一样的景色，那些各式各样的房子。澳洲土是土，但房子着实是我喜欢的风格，都是自家修的，每个都有自己的特色，有很多煞是好看。





一路上跑着步，听着歌，看屋子，看鸟儿，看树木，看月亮，腿好像也变轻了，心情愉悦，好像突然就get到了跑步的乐趣。到了下个路口，惊喜地发现是我认识的地方，是上次坐巴士下错站之后摸回家的路上路过的洗车店，于是安心。

在路口凭着记忆左转，暮色下的公路蓦然撞进眼里，落日残霞，老树寒鸦，然而词穷的理科生只会说：好美啊~

家附近的小阶梯，成功回来啦~

开端很美好，希望能坚持，明天挑另一个路口拐进去，多看些不一样的风景~



20-04-06

今天出门晚了，因为还是没适应夏令时制结束后澳洲的天黑得早了许多，于是只敢沿着上次同样的路线跑了一圈，不过跑得明显比上次轻松了，开心嘞~

回到家这边时天已经很黑了，跑着步被头顶电线上爬动的生物吓了一跳，好像是房东小姐姐说过的负鼠，忒大一只了，我生怕它掉下来。

跑完一看数据：我心跳这么快过？！

出汗的感觉真好。



20-04-07

今天特意早些出了门，出门的时候留守的波比日常委屈哈哈哈哈。

因为天儿早，所以不想再跑重复的路线，重新挑了个路口拐进去，一路看了很多不同的房子。




拐了许多次弯之后发现到了一条平常跑的路上，反正天不黑就胆子大，于是特意再次拐到了不熟的路上，结果跑了一会儿发现路过一户人家，门口有着昨天看到过的蹦床，这真是弯弯绕绕又回来了哈哈哈哈。




每次停下来拍人家的房子时总怕被人打是怎么回事orz

现在跑步同时使用小米手环和Nike Run Club，看着运动记录的数据，特别有成就感，耳机里的运动教练一直激情澎湃地夸着你优秀啊真优秀，坚持跑就很优秀，哈哈哈哈那希望我一直优秀~


20-09-14

一转眼这就到下半年了，在澳洲百分之八十的时间都在家里蹲，除了出门跑步和采购就没了其他户外活动。

由于禁令的限制，也不敢乱跑，索性一直都在重复的路线上循环，没了新鲜的路线，路边的风景自然也就不再新鲜。有意思的是，这路线上有那么一户人家，我每天规律地运动，他家每天规律地做饭，结果就是我总是规律地在路过他家时闻见极香的饭菜味道，然后下意识地放慢脚步。那味道，嗯，我断定这是一家华人住户。

跑步的时间段随着日头落下的时间变化，从黄昏到清晨再到黄昏，不过至今最远也只试过两次八公里而已，且跑完都会膝盖疼痛，吓得我只敢老老实实不超过六公里的距离。这样下来，虽是在五公里之内的跑速上有了些提高，但每次看见老师跑步的超长路线图我都还是免不了觉得佩服且羡慕，还是想要尝试更远的距离。

当初老师教我跑步要三天打鱼两天晒网，半年之后再开始第一个五公里，我现在算是基本完成了，也许可以试试进阶了。今天在软件的指导下制定了一个十公里跑步突破计划，接下来争取循序渐进，把人生第一个十公里留在澳洲的土地上~



20-09-24

跑步也有好几个月了，随着最近配速慢慢的提升，同时也习惯了跑五公里，一个问题开始令我疑惑：为什么不管我跑五公里六公里，总是最后一公里突然开始感觉到累呢？

之前跑五公里的时候，软件会提示“还剩最后一公里”，所以我经常前四公里觉得跑得很顺畅，也比较轻松，是听到提示之后突然觉得疲劳，身体和心里都觉得累得很明显，无法保持之前的跑速。如此几次之后我以为是我的体力只够我跑四公里。但奇怪的是当我提量到六公里时，同样是前五公里轻松，第六公里突然开始迈腿艰难，于是我又以为是临近成功的提示让我感到松懈了。然鹅，当我设置目标为四公里，跑四公里当然不会让我觉得很累（哈哈，果然有进步，以前跑三公里都会累的），当软件提示目标完成后我会继续跑，直至完成六公里，但这时，同样又是第五公里轻松完成，而在第六公里开始需要艰难坚持，咋肥四啊？？

暂定一个解释：我目前的能力是能较轻松地跑完五公里，且我在心里估摸着自己快要完成目标的时候就松懈了，即使软件没有提醒我。留待能跑更远的时候更新答案吧。

20-10-31

今天想记个流水账。

事情要从今天的LSD说起，今日份的长里程慢跑，因为不想枯燥重复路线，天光尚早，就又开始了新区域的探索。随心而跑，也不怕迷路。倒确实发现了新的公园，而且好像很大的样子，本来想跑进去，结果发现路是砂石路，入口的碑还提醒小心蛇，我怂了。于是沿着公园的边跑起来，意外在路边一户人家的门口发现了装着糖果的小盆。很想拿，但我毕竟不是个小朋友QAQ，也不懂国外风俗，于是没敢伸手，只是拍了照就继续跑走了。谁知返程时由于记错了路，误打误撞在错误的路上又遇见了一个糖果盆盆，这下我对糖的渴望彻底被勾起来了（大概是因为看见了盆里的棉花糖？我仿佛已经能想象出那软绵香甜的味道）。还是怂着没拿，但跑回去的一路上都在记挂着那糖（其实心里有点想着拿了盆盆里的糖就当我也在国外过了一把万圣节了），我甚至还想干脆去超市买一包棉花糖或者大白兔奶糖好了。但，我毕竟是个减肥的人，如果买了一包，说不定一包今天就被干完了。于是我想，要不去老师家门口看看有没有糖果盆？有的话正好只拿一颗过过瘾。






又是几公里的路程，跑到老师家，发现并没有糖（事实上，今天十几公里中也只有那两家有糖，被我遇上很难得了）。想想不甘心，遂把糖果盆的照片发给老师，问：不是小朋友可以拿糖吗？得到肯定回答后，我返程跑了几公里，就为了拿两颗糖。可惜的是有一家的糖出乎我意料地很快就被拿得所剩无几，我没能赶上我想吃的那块QAQ不过本小朋友还是很开心了，虽然是戴着口罩厚着脸皮拿的哈哈哈哈，甚至在心里安慰自己“我的身高可以支持我冒充一回小朋友的”，但好歹是吃上了万圣节的糖果，真是甜甜的幸福感啊~



今日感悟。小公园里的跑道，我路过五次了，每次都想着那坡上的岔路是通向哪里呢？但看样子总以为是通向类似公园里的富户的私人住宅区，于是没敢去看。今天想着去老师门口找糖，经过那条路的时候，恍然发觉，我以为的富户住宅区，就是老师家那片房区啊。人啊果然还是要大胆探索，要不是我畏缩，不敢走那十几步，这件事早就应该被发现了QAQ


相机拍不出风景的美丽，只能尽力用眼睛留下记忆。
